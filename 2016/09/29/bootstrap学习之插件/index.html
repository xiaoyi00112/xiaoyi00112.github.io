bootstrap学习之插件

前端
  • bootstrap
  • 前端

Bootstrap全部知识点整理思维导图
插件函数学习思维导图
来一款小游戏放松一下2048

概述

  1. Bootstrap提供给了用户十几个基于jQuery的插件函数，可以一次性全部引入，也可以进行单个插件函数的引入。

  2. 调用方式有两种

    • data属性进行调用，关闭方法

      $(document).off('.data-api')

    • 编程方式的API

      $('.btn.danger').button('toggle').addClass('fat')

下拉菜单

  1. data属性方式：给链接或button添加data-toggle=”dropdown”
  2. js方法调用：$(‘.dropdown-toggle’).dropdown(),但是仍然需要加上data-toggle=”dropdown”。特殊效果是使下拉菜单一上来就显示$(‘.dropdown-toggle’).dropdown(‘toggle’)

标签页

  1. data方式：data-toggle=”tab” or data-toggle=”pill”;

    • js方式：加入js代码（each tab needs to be activated individually）

      $('#myTab a').click(function (e) {
  e.preventDefault()
  $(this).tab('show')
})
    • Fade effect:渐变效果;.fade .in

警告框

  1. .alert .alert-warning
    小差号：&+times;加上.close会浮动到右边，注意：带符号的元素应该放到第一个子元素的位置上。
  2. 官方文档有一个”.sr-only”的属性，此属性是专为屏幕阅读器准备，屏幕阅读器外，其他设备上隐藏该元素，帮助残障人士设计。
  3. 实现关闭效果推荐用data-的方式。具体实现方式是给button标签增加 data-dismiss=”alert” 自定义属性。
  4. 如果给父元素添加 .fade 和 .in 类，那么元素关闭时呈现淡入淡出效果

工具提示、弹出框和摸态框

三种形式的弹出框：1.工具提示（tooltip）；2.弹出框（popover）;(3).模态(modal)弹出框–重点；

  1. 工具提示：加一个data-toggle=”toolti” title=”提示文字” data-placement=”弹出位置”，还需要加上一行js代码进行初始化：

    $(function(){
  $('[data-toggle="tooltip"]').tooltip();
})

  2. 弹出框：data-toggle=”popover” title=”标题” data-content=”弹出内容” ，同样也需要js代码进行初始化：

    $(
  $('[data-toggle="popover"]').popover();
)

  3. 模态弹出框：

    • 模态(Modal)子窗体：由某个父窗体打开，只要子窗体不关闭，父窗体无法获得输入焦点；且父子窗体可以互相传递数据。
    • 具体实现：摸态框固定定位，初始时一般不显示，将模态框放到HTML代码的文档最高层级，推荐放到body的最后，直接作为body的子元素。
      c:data-toggle=”modal” herf=”目标元素”
    • button触发：data-toggle=”modal” data-target=”目标元素”

    • 模态框的结构：

      <div class="modal">--全屏的遮罩层
  <div class="modal-dialog">-->宽高、定位
    <div class="modal-content">-->背景色、边框、倒角、阴影
      <div class="modal-header"></div>
      <div class="modal-body"></div>
      <div class="modal-footer"></div>
    </div>
  </div>
</div>

    • 可以使用data-dsimiss=”modal”来关闭模态框。

Collapse折叠效果

  1. 使用a标签和button触发一个普通折叠效果：data-toggle=”collapse”;class里面有in的时候默认为显示，没有默认为隐藏
  2. 手风琴折叠效果：需要借助面板组件配合实现。需要.panel-group 可以吧导航信息放到

  3. 响应式导航条：类似Bootstrap官网折叠导航栏效果

    <div  class="navbar navbar-default">
  <div class="container">
    <div class="navbar-header">
      <a href="#" class="navbar-brand">TARENA</a>
      <a href="#myNBA" class="navbar-toggle" data-toggle="collapse">
        <span class="icon-bar"></span>
        <span class="icon-bar"></span>
        <span class="icon-bar"></span>
      </a>
    </div>
    <div id="myNBA" class="collapse navbar-collapse">
      <ul class="nav navbar-nav">
        <li><a href="#">首页</a></li>
        <li><a href="#">首页1</a></li>
        <li><a href="#">首页2</a></li>
      </ul>
    </div>
  </div>
</div>

Carousel广告轮播。旋转木马

  1. 首先需要一个carousel的父容器div，slide是带滑动效果，id为了之后的点击滑动，data-ride=”carousel”,data-interval=”设定时间”；
  2. 然后内部建立carousel-inner,内部建立一个.item的div，div中一次可以放img图片和加入图片的文字，加入文字需要再建立一个.carousel-caption的容器div，里面放入内容
  3. 左右点击用a标签，href制定为carousel父容器的id，data-slide=”prev/next”
  4. 滑动可以用ul和li来制作，ul的class为carousel-indicators,li设置data-target=”ID” data-slide-to=”图片位置”。
  5. 注意：放在carousel-inner外的元素不随广告滚动

Affix附加导航

给每个标题加上id,然后添加一个data-spy=”affix”的导航元素。元素内部还定义有一个data-offset-top=””的属性。

  • data-spy=”affix” 定义导航元素。
  • data-offset-top:从网上找了好多答案，说的都挺模糊其实这个属性还可以写成data-offset和data-offset-bottom,这两种形式，data-offset-top的意思是当你鼠标往上滚动的距离为多少的时候，这个导航条会随着页面动，也就是说，如果你把这个值设为0，你一往下滚动鼠标，导航条就随着页面动，如果设为100，那么你的鼠标滚动100px的时候导航条才会动，并且适应屏幕。

滚动监听（快要胜利了啊）
