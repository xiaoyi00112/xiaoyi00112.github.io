bootstrap学习之组件

前端
  • bootstrap
  • 前端

学习思维导图

一、Glyphicons Halflings字体

Bootstrap提供了大约250个图标，作为服务器端字体，供浏览器客户端下载使用。

强调：必须和css在同级目录下，并且名字必须为fonts，因为Bootstrap里面对这套字体进行了定义，在css文件内部使用的名字就是fonts。下面是css中声明服务器端字体。
放到任意内容为空的元素中，并且一个元素只能包含一个图片。

@font-face {
  font-family: 'Glyphicons Halflings';

  src: url('../fonts/glyphicons-halflings-regular.eot');
  src: url('../fonts/glyphicons-halflings-regular.eot?#iefix') format('embedded-opentype'), url('../fonts/glyphicons-halflings-regular.woff2') format('woff2'), url('../fonts/glyphicons-halflings-regular.woff') format('woff'), url('../fonts/glyphicons-halflings-regular.ttf') format('truetype'), url('../fonts/glyphicons-halflings-regular.svg#glyphicons_halflingsregular') format('svg');
}

二、下拉菜单

使下拉菜单更具交互性。
首先需要将下拉菜单触发器和下拉菜单都包含到.dropdown中，或者声明一个position: relative;的元素。.caret为下拉小三角样式类.dropdown-menu和role=”menu”是下拉菜单的父元素，包含下拉的每项。

  1. 为下拉菜单设置对齐方式
  2. 为下拉菜单设置标题
  3. 为下拉菜单设置分割线

注意：根据Bootstrap官方文档中提供的例子里有role和aria-labelledby属性，这两个属性是专门为无障碍阅读软件而设计。还有一个属性是aria-label。
具体用法可以参考信息无障碍产品联盟

三、按钮组

把一组按钮放到一个父容器中，并给父容器添加css样式,这样按钮就加入到了一组，并形成按钮组的样式。实现的主要方式是通过复杂选择器对按钮组中的按钮进行样式的设置。

  1. 基本实例，.btn-group
  2. 按钮组工具栏,.btn-toolbar
  3. 按钮组尺寸,.btn-group-*(lg,,sm,xs)
  4. 按钮组嵌套。只需要两个.btn-group嵌套就可以了
  5. 垂直排列。.btn-group-vertical
  6. 两端对齐。类似变成行内块，填充一整行。.btn-group-justified

四、按钮式下拉菜单

把任意一个按钮放入 .btn-group 中，然后加入适当的菜单标签，就可以让按钮作为菜单的触发器了。

  • 最外层为.dropdown样式类
  • 触发按钮，需要添加.dropdown-toggle样式类和data-toggle=”dropdown”
  • ps:调用Bootstrap插件函数的两种方式，1.利用传统的JS变成方式：$(..).插件函数(..);2.使用data-（）扩展属性<元素 data-Xx-“Yy”>第二种方法在底层的底层已经封装成了类似$(‘[data-Xx=”Yy”]’).XXXX…
  • 下拉菜单为.dropdown-menu样式类和role=”menu”

注意：需要Bootstrap版本中含有下拉菜单插件，最新版的一般都含有，只需要将全部插件直接引入即可。

  1. 单按钮式下拉菜单；绑定触发样式类.dropdown-toggle和data-toggle=”dropdown”就可以触发了
  2. 分裂式按钮下拉菜单；上下两个按钮，一个名字，一个触发事件
  3. 按钮尺寸；btn-(lg,sm,xm)
  4. 向上弹出式菜单；给父类添加.dropup即可

五、输入框组

通过文本输入框input（只支持文本输入框）前面和后面加入文字或按钮，丰富输入框的样式。设置方式是为输入框和文字的包裹父元素设置.input-group。然后用文本元素包裹文字，并给文本元素设置.input-group-addon,input元素设置为.form-control。如果用表单和栅格系统，则需要将输入框嵌入其内部。

  1. 基本实例：

    <div class="input-group">
  <span class="input-group-addon">@</span>
  <input type="text" class="form-control" placeholder="Username">
</div>

<div class="input-group">
  <input type="text" class="form-control">
  <span class="input-group-addon">.00</span>
</div>

<div class="input-group">
  <span class="input-group-addon">$</span>
  <input type="text" class="form-control">
  <span class="input-group-addon">.00</span>
</div>

  2. 输入框尺寸;为父元素设置input-group-(lg,,sm)

  3. 嵌入单选框和复选框；将单选框或复选框放入.input-group-addon类的元素内
  4. 添加额外的按钮；，将按钮放入.input-group-btn类的元素内
  5. 添加下拉菜单：同上，将按钮放入.input-group-btn类的元素内
  6. 添加分裂式下拉菜单：同上，将按钮放入.input-group-btn类的元素内

六、导航

共同依赖一个.nav类，必须首先进行添加

  1. 标签页：.nav .nav-tabs
  2. 胶囊式标签页：.nav .nav-pills，垂直方式：.nav-stacked
  3. 两端对齐的标签页（已废用）.nav-justified
  4. 禁用的链接：.disabled，只禁用a标签
  5. 添加下拉菜单：只需要把下拉菜单添加到标签页的具体项中（a标签的后面），

七、导航条

  1. 表单：.navbar-form垂直对齐效果
  2. 按钮：.navbar-btn
  3. 文本：.navbar-text
  4. 非导航的链接：.navbar-link
  5. 组件排列：.navbar-right,navbar-left
  6. 固定在顶部：.navbar-fixed-top
  7. 固定在底部：.navbar-fixed-bottom
  8. 静止在顶部：.navbar-static-top
  9. 反色导航条：.navbar-inverse

八、路径导航：.breadcrumb

九、分页

  1. 分页：.pagination很漂亮的分页效果
  2. 禁用和激活状态:.disabled .active
  3. 尺寸：.pagination-(lg,,sm)
  4. 翻页：.pager
  5. 对齐链接：.previous .next

十、标签

.label .label-(default,primary,success,info,warming,danger)

十一、徽章

.badge

十二、巨幕

.jumbotron:放到父元素中，对巨幕中的内容包裹

十三、页头

page-header:放到包裹元素中

十四、缩略图

.thumbnail

十五、警告框

  1. .alert .alert-(success,into,warning,danger)
  2. .alert .alert-warning .alert-dismissible
  3. 警告框中的链接 .alert-link

十六、进度条

  1. 基本样式：父元素.progress 子元素.progress-bar 设置span为.sr-only
  2. 带提示标签的进度条:移除.sr-only
  3. 根据情境变化效果:.progress-bar-(success,info,warning,danger)
  4. 条纹效果:.progress-bar-striped
  5. 动画效果：.active
  6. 堆叠效果：多种样式的进度条放到一个.progress中

十七、媒体对象

  1. .media .media-list
  2. .media-left .media-middle
  3. .media-body .media-heading

十八、列表组

  1. 基本实例：.list-group .list-group-item
  2. 带徽章的列表组：在徽章（.badge）加在列表项的后面
  3. 链接列表:用div和a分别替换ul和li
  4. 禁用:.disabled
  5. 情景类：可以在列表项中使用.list-group-item-success等和.active

十九、面板

.panel .panel-heading .panel-title .panel-body .panel-footer panel-(default,primary,success)
