创建模板

想要简历一个bootstrap网页，首先需要简历bootstrap网页模板，下面是官网模板简化

<!DOCTYPE html>
<html lang="zh-cn">
  <head>
    <meta charset="utf-8">
    <!-- 跨IE浏览器版本兼容性问题 -->
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <!-- 媒体查询 -->
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>Bootstrap 101 Template</title>

    <!-- 引入bootstrap的css样式文件 -->
    <link href="css/bootstrap.min.css" rel="stylesheet">

    <!-- 两个非常经典的解决IE8以下浏览器兼容问题 -->
    <!-- htmlshiv.js支持H5新标签 -->
    <!-- respond.js支持响应式网页 -->

    <!--[if lt IE 9]>
      <script src="http://cdn.bootcss.com/html5shiv/3.7.2/html5shiv.min.js"></script>
      <script src="http://cdn.bootcss.com/respond.js/1.4.2/respond.min.js"></script>
    <![endif]-->
  </head>
  <body>
    <h1>你好，世界！</h1>

    <!-- bootstrap里的插件函数是基于JQuery写的 -->
    <script src="http://cdn.bootcss.com/jquery/1.11.1/jquery.min.js"></script>
    <script src="js/bootstrap.min.js"></script>
  </body>
</html>

配置Bootlint检测使用Bootstrap的css使用错误

Bootlint是Bootstrap官方所支持的HTML检查工具，可以自动检测常见html错误，或者Bootstrap组件需要固定的DOM结构。在学习和开发阶段可以避免一些简单错误。
建立过程是在Bootlint官网上复制下以下代码，放到html文件中body的结束标签之前。从代码中可以看出，Bootlint是创建了一个script标签并加载到body的DOM树中，然后通过回调函数s.onload调用方法，此方法会生成一个检测报告。

<script>(function () {
  var s = document.createElement("script");
  s.onload = function () {
    bootlint.showLintReportForCurrentDocument([]);
  };
  s.src = "https://maxcdn.bootstrapcdn.com/bootlint/latest/bootlint.min.js";
  document.body.appendChild(s)
})();</script>
